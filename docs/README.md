## üìñ What is Air Space Control Website?

An institutional portal for air traffic control units that simulates real-time information and statistics about air traffic operations. Built with Laravel/PHP backend and Vue.js frontend, it provides:

-   Real-time flight tracking
-   Control tower management
-   Radar approach control
-   Operational statistics and reporting
-   User authentication and authorization
